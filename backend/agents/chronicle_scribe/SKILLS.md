# Chronicle Scribe Skills

## Output Formats

### Strategic Report
```markdown
# Case Analysis: [Title]
## Executive Summary
## Key Findings
## Recommended Strategy
## Action Plan
## Risk Assessment
## Appendices
Pre-written Emails
Lawyer engagement
Insurance claim formalization
Counterparty communication
Regulatory complaint
Visual Diagrams
Mermaid Gantt (timeline)
Mermaid Graph (relationship map)
Risk matrix (impact vs probability)
Contact List
Table
Name	Role	Organization	Contact Method	Priority
```

### Style Guidelines
- Active voice, present tense for findings
- Past tense for historical events
- Future tense for recommendations
- Bullet points for scannability
- Bold for critical warnings
