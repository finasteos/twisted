# Smart Library: Context Analyzer capabilities

This package contains the specific tools and skills available to the Context Analyzer.

## Capabilities
- **Semantic Extraction**: Ability to pull exact dates, names, and monetary values from dense text.
- **Timeline Reconstruction**: Skill to map disparate events into a single linear progression.
- **Sentiment Profiling**: Ability to gauge the emotional tone of communications (e.g., adversarial, cooperative, confused).
- **Document Chunking Awareness**: Ability to understand partial context retrieved from semantic vector stores (RAG).
